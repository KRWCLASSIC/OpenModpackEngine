If you want to force old OMEmodule to work with current version here is internal version numbering you can use (used by "build-on-ome" variable)

Used by Alpha Versions of OME:
0 - "Alpha0"
1 - "Vo1d Eater" (Without ASCII support in OMEmodule builder)
1 - "Van1lla Cream" (With ASCII support in OMEmodule builder)
2 - "AZA2EL Soulstealer" (Basic "additions" implementation)
3 - "Blood R3aper" (Support for including own loader)

Used by Beta Versions OME:
4 - "Hell V4riation" (Not released yet)
